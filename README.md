# logicgate
This is a mountain-themed website built using pure HTML and CSS as part of my frontend development learning journey.

🚀 Key Features
Semantic HTML: Structured with semantic tags like <header>, <main>, and <footer> for better accessibility.

CSS Grid: Implemented a responsive 12-column grid system for a clean layout.

Modern UI: Focused on a professional dark theme with blur effects and card styling.

🛠 Tech Stack
HTML5

CSS3
